Hello! A liitle bit about the game:
In this Pacman game we used java script, jquery and HTML5 elements and did not use any templates.
We used jQuery at the login and signin form validations. Java script has used us mainly for the purpose of building and managing the game board.
Working on this game was very enjoyable. The main challenge during the work was in the javascript validations and the game logic.
We added 3 extra functions:
1. The user can choose the ghosts speed
2. The user can choose number of points needed to win
3. If the pacman has less than 3 lives, a one-time extra life in the shape of an heart will appread in a random time.

~*~Some Instructions~*~
This is a version of the popular Pacman game!
To move the pacman use your keyboard arrorws. Everytime you hit a ghost, you lose life. If you see a heart on the board - that's an extra life bonus!
A cookie is a food that worth 50 points. To finish the game you need to survive the entire time or finish taking all the points. To play you have to register or login first.
At any moment you can press the main logo to return to the main page.
When you start the game, a setting menu will appeared to let you choose your game preferences.
Attention: Pressing the about button during the game won't pause the game!

Hope you will enjoy playing our game!
